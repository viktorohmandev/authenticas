INTEGRATION

Enkel API-integration
för alla tekniska stackar

Vårt API är designat för att vara enkelt att använda. Verifiera inköp med ett enda API-anrop, oavsett er teknikstack. 

curl -X POST https://api.authenticas.se/verifyPurchase \
  -H "Authorization: Bearer DIN_API_NYCKEL" \
  -H "Content-Type: application/json" \
  -d '{
    "email": "anstald@foretag.se",
    "amount": 2500,
    "currency": "SEK"
  }'
